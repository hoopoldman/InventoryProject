Interface Store -> 함수만 구현, Mystore에서 기능을 구현함..

Item 클래스 
재고가 가지는 속성 -> price(물품 가격), name(물품 이름), num(재고량), company(회사), site(물품을 산 사이트..쿠팡 이런거) / 각 멤버변수별로 getter/setter 함수를 정의하여 접근함
생성자 정의-> 종류를 여러개 한 이유..사용자가 사이트, 회사, 가격은 입력안할 수 있기 때문에 그 경우를 대비하여 정의함..하지만, 이름과 수량은 필수적으로 입력받도록 할것..이는 예외처리를 추후에 만들어 방지할 예정
